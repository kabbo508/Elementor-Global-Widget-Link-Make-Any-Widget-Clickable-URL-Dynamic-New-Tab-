# Elementor Global Widget Link

Make any Elementor widget fully clickable, just like a card or block, without editing HTML or breaking layouts.

This lightweight plugin adds a “Whole Widget Link” option inside the Advanced tab of all Elementor widgets, allowing you to assign a URL to the entire widget area.

## Key Features

- Works with ALL Elementor widgets automatically
- Add full widget clickable link
- Supports Dynamic Tags
- Open in new tab option
- Nofollow support
- Safe implementation (no HTML breaking)
- Compatible with Elementor Free & Pro
- Lightweight and fast, no performance overhead

## Why This Plugin?

Elementor does not provide a native way to make an entire widget clickable. Normally, you need custom code per widget, which is inefficient and inconsistent.

This plugin provides a clean, scalable solution using Elementor’s internal wrapper system.

## How It Works

- Injects a link control into all widgets (Advanced tab)
- Adds link attributes to Elementor wrapper
- Uses a safe JavaScript handler to enable full widget click
- Prevents conflict with inner buttons, links, and form fields

## Installation

1. Download the plugin ZIP
2. Go to WordPress Admin → Plugins → Add New
3. Upload ZIP and activate
4. Edit any Elementor widget
5. Go to Advanced → Whole Widget Link

## Use Cases

- Clickable service cards
- Pricing boxes
- Course blocks
- Portfolio items
- CTA sections
- Landing page UI blocks

## Compatibility

- Elementor Free
- Elementor Pro
- Works with most themes
